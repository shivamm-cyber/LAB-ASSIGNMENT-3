# Lab Assignment 3 - Responsive Personal Portfolio

Fully responsive portfolio built with **pure HTML5 & CSS3** (no frameworks).

## Features
- Modern glassmorphism + gradient design  
- Fully mobile & tablet responsive  
- CSS Flexbox + Grid layout  
- Smooth hover effects & scroll animation  
- Clean & professional code structure (3 files)

## Tech Stack
- HTML5  
- CSS3 (Flexbox, Grid, Variables, Animations)  
- Font Awesome Icons  
- Google Fonts
