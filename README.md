# LAB-ASSIGNMENT-3  
### Responsive Personal Portfolio

Fully responsive portfolio built with **pure HTML5 & CSS3** (no frameworks).

## Features
- Modern gradient + glassmorphism design  
- 100% mobile & tablet responsive  
- CSS Flexbox + Grid layout  
- Smooth hover effects & scroll animations  
- Clean professional code (3 files structure)

## Tech Stack
- HTML5  
- CSS3 (Flexbox, Grid, Variables, Animations)  
- Font Awesome Icons  
- Google Fonts
